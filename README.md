This repo aims to demonstrate how to use Ansible to manage Docker Swarm cluster, using Docker-in-Docker (DinD) to simulate a cluster of Docker hosts, connected to a Docker network (automatically generated by Docker Compose)

Since Ansible required Python to be installed on the target hosts, we need to install Python on the Docker containers. We also need to install `sudo` to run commands as root. That's why tweaked Docker DinD are built (see Dockerfile)
